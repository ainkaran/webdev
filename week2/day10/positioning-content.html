Position Property

position: static;

Note: this is the default

position: fixed;

We can give it coordinates relative to the top left corner with top and left.

position: absolute;

It's taken out of the flow, but it still moves with scrolling. You can think of its position as based on top left plus scroll coordinates

We can move it around with the properties top, bottom, right and left.

We can move it around with the properties top, bottom, right and left.

position: relative;

By itself, relative has little effect on the container.

Children positioned absolutely now use the relative container as a base for their coordinate system.

Absolute is most often used inside a relative element.

z-index: [any integer];

Elements positioned with absolute and fixed are normally stacked on top of each other based on the order they appear in HTML. First on bottom and last on top.

You can change that by using z-index. Changes the z coordinate.
